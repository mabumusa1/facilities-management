# {{ page.title }}

*Create a visitor invitation with a QR code so your guest can enter the community at the gate — no phone calls required.*

## Who this is for

**Residents** who want to pre-register an expected guest, delivery, or service visit.

## Before you start

- You must be signed in as a Resident.
- Your community must have Visitor Access enabled. Contact your Property Manager if the **My Visitors** section is not visible in your account.

## Steps

### Register a visitor

1. From the navigation menu, go to **My Visitors**.
2. Tap **Register Visitor** (top-right, or the button in the centre of the empty state).
3. Fill in the form:
   - **Visitor Name** — enter the visitor's full name. Arabic names are accepted.
   - **Purpose** — choose one: **Visit**, **Delivery**, **Service**, or **Other**.
   - **Expected arrival** — pick the date and time you expect the visitor to arrive.
   - **Visitor phone (optional)** — add a phone number if you want it on record.
4. Tap **Generate QR Code**.
5. The **Invitation Created** screen opens and shows the QR code, visitor details, the status badge, and the time the code is valid until.

### Share the QR code with your visitor

1. On the **Invitation Created** screen, tap **Share QR**.
2. Your device's share sheet opens — send the code via WhatsApp, SMS, email, or any other app.
3. Your visitor shows the QR code to the gate officer on arrival. The officer scans it to verify and record entry.

### View and manage your invitations

1. Go to **My Visitors** at any time to see all your invitations.
2. The list is split into two sections:
   - **Active** — invitations that are pending or within the valid arrival window.
   - **Past** — invitations that have been used, expired, or cancelled.
3. Tap **View QR** on any active invitation to open its detail screen and share the code again.

### Cancel an invitation

1. On the **My Visitors** list, find the invitation you want to cancel.
2. Tap **Cancel Invitation**.
3. A confirmation dialog shows the visitor's name, purpose, and expected arrival time.
4. Tap **Cancel Invitation** in the dialog to confirm. Tap **Keep Invitation** to go back.
5. Once cancelled, the QR code is immediately invalidated. The gate officer cannot admit the visitor with that code.

## What you'll see

After you create an invitation you see:

- The visitor's name and the visit purpose.
- The invitation **status badge** — starts as **Active** once saved.
- The QR code image, ready to share.
- **Valid until** — the exact time the code expires (set by your community admin; default is 24 hours from creation).

On **My Visitors**, each card shows the visitor name, purpose, expected arrival, and a status badge:

| Status | Meaning |
|---|---|
| Active | The code is valid and will be accepted at the gate. |
| Used | The gate officer has already scanned this code. |
| Expired | The valid-until time has passed. Create a new invitation if the visitor still needs to come. |
| Cancelled | You cancelled this invitation. The code is no longer usable. |

## Common issues

- **"My Visitors" is not in my menu** — Visitor Access may not yet be enabled for your community. Contact your Property Manager.
- **The gate officer says the code is not working** — check the invitation status on **My Visitors**. If it shows **Expired**, create a new invitation. If it shows **Active**, ask the gate officer to scan again or verify the QR image loaded fully on the visitor's screen.
- **I shared the link but the QR image does not appear** — tap **View QR** on the invitation card in **My Visitors** and share it again directly from that screen.
- **The visitor arrived but the status still shows Active** — the gate officer records entry by scanning the code. Until they do, the status stays Active.
