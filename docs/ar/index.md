# مركز المساعدة

أدلة وإرشادات وملاحظات الإصدار لمنصة إدارة العقارات.

## حسب الدور

- **السكان** — حجز المرافق، تقديم طلبات الخدمة، دفع الإيجار.
- **مُلّاك الوحدات** — متابعة الإيجارات، عرض الحسابات، تسليم الوحدات.
- **مديرو العقارات / المسؤولون** — إدارة المجتمعات، اعتماد الإيجارات، تشغيل العمليات.
- **مستخدمو السوق** — إدراج الوحدات، تصفح القوائم، تقديم العروض.
