# Help Center

Guides, how-tos, and release notes for the real estate management platform.

## By role

- **Residents** — book facilities, file service requests, pay rent.
- **Unit Owners** — track leases, view accounting, hand over units.
- **Property Managers / Admins** — manage communities, approve leases, run operations.
- **Marketplace users** — list units, browse listings, make offers.
