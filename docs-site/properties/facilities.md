# Facilities & Amenities

**Facilities** are shared community spaces and amenities — swimming pools, gyms, meeting rooms, parking areas, and similar resources that residents can book.

## Viewing Facilities

Navigate to **Properties → Facilities** or open a community and click **Facilities**.

## Adding a Facility

1. Navigate to **Settings → Facilities** or use **Directory → Add Facility**
2. Click **+ Add Facility**
3. Fill in:

| Field | Description |
|-------|-------------|
| **Name** | Facility name (e.g., "Main Swimming Pool") |
| **Category** | Pool, Gym, Meeting Room, Parking, etc. |
| **Community** | The community this facility belongs to |
| **Capacity** | Maximum number of people at once |
| **Booking Hours** | Available hours for booking |
| **Requires Approval** | Whether bookings need admin approval |

4. Add photos
5. Click **Save**

## Facility Details

The facility details page shows:
- Current and upcoming bookings
- Booking statistics
- Configuration settings

## Enabling Facility Bookings

Residents can only book facilities if the **Facility Bookings** feature is enabled. Go to **Settings → Facility Bookings** to configure this.

See [Facility Bookings](/facilities/bookings) for how residents make and manage bookings.
