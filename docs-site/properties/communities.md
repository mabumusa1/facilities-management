# Communities

A **community** is the top-level container in your property portfolio — typically a residential complex, gated community, or commercial development.

## Viewing Communities

Navigate to **Properties → Communities** to see all communities in your account.

The list view shows each community's name, total buildings, total units, and current status.

## Creating a Community

1. Click **+ New Community**
2. Fill in the required fields:

| Field | Description |
|-------|-------------|
| **Name** | The community's display name |
| **Country** | Country where the community is located |
| **City** | City or municipality |
| **District** | District or neighborhood |
| **Currency** | Default currency for financial transactions in this community |

3. Click **Save**

The community is created with **Active** status by default.

## Bulk Upload

To create multiple communities at once:

1. Navigate to **Properties → Communities**
2. Click **Bulk Upload**
3. Download the provided Excel template
4. Fill in your community data
5. Upload the completed file

Any validation errors will be shown with row numbers so you can correct them.

## Community Details

Click any community name to open the details page. From here you can:
- View all **buildings** within the community
- See a summary of **units**, **leases**, and **occupancy**
- Access **facilities** (pools, gyms, parking areas)
- Manage the **community directory**
- View **announcements** scoped to this community

## Editing a Community

Open the community details and click **Edit**. Update the fields and click **Save**.

## Deactivating a Community

If a community is no longer in use, open it and set the **Status** to **Inactive**. Inactive communities are hidden from most views but their data is preserved.

::: warning
Deactivating a community does not cancel active leases or service requests. Handle those separately before deactivating.
:::
