# Units

A **unit** is an individual space that can be leased or sold — an apartment, villa, office, or retail space.

## Viewing Units

Navigate to **Properties → Units** to see all units across your portfolio, or drill into a specific building to see its units.

Use the filters to narrow by:
- **Status** (Available, Rented, Sold, Sold & Rented)
- **Type** (Apartment, Villa, Studio, Office, etc.)
- **Community** or **Building**
- **Floor**

## Creating a Unit

1. Go to the building you want to add a unit to
2. Click **+ New Unit**
3. Fill in the required fields:

| Field | Description |
|-------|-------------|
| **Unit Number** | Unique identifier within the building |
| **Floor** | Floor number |
| **Category** | Residential or Commercial |
| **Type** | Apartment, Studio, Villa, Office, etc. |
| **Area (sqm)** | Total unit area |
| **Market Rent** | Expected monthly rental amount |
| **Market Price** | Expected sale price (if applicable) |
| **Bedrooms / Bathrooms** | Room counts (residential units) |

4. Attach any photos or documents
5. Click **Save**

### Bulk Upload Units

Download the Excel template from **Properties → Units → Bulk Upload**, fill it in, and re-upload to create multiple units at once.

## Unit Details

The unit detail page is the central hub for a unit's information:

### Overview Tab
- Current status and occupancy
- Key specs (area, floor, type)
- Market rent and price

### Lease Tab
- Active lease details
- Lease history

### Transactions Tab
- Rent payments and outstanding balances

### Service Requests Tab
- Open and resolved maintenance requests for this unit

### Documents Tab
- Attached photos, floor plans, and contracts

## Unit Statuses

| Status | Description | What You Can Do |
|--------|-------------|-----------------|
| **Available** | No active lease | Create a new lease, list on marketplace |
| **Rented** | Has an active lease | View lease, submit service requests |
| **Sold** | Ownership transferred | View transaction history |
| **Sold & Rented** | Sold but also occupied | Manage both ownership and tenancy |

## Marketplace Listing

To list a unit for rent or sale on the marketplace:
1. Open the unit
2. Click **Marketplace** tab
3. Click **Add to Marketplace**
4. Set the listing type (For Rent / For Sale), price, and description
5. Upload photos
6. Click **Publish**

## Editing a Unit

Open the unit and click **Edit**. All fields can be updated except Unit Number (contact your administrator to change the unit number).
