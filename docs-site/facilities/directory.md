# Community Directory

The **Community Directory** provides residents and staff with a browsable view of the community's buildings, facilities, and key contacts.

## Accessing the Directory

Navigate to **Directory** in the left menu (or **Dashboard → Directory**).

## What's in the Directory

### Buildings
- List of all buildings in the community
- Key building information (location, floors, amenities)

### Facilities
- Community amenities (pool, gym, parking, etc.)
- Facility details, operating hours, and booking button

### Owners
- Property owners associated with the community

## Adding Directory Entries

Admins can add or update directory information:

1. Navigate to **Directory → + Add Entry** (or **+ Add Facility**)
2. Fill in the details for the building, facility, or contact
3. Add photos
4. Click **Save**

## Directory Documents

Community-wide documents (such as rules and regulations, community guidelines, and emergency contacts) can be uploaded to the directory:

1. Navigate to **Directory → Documents**
2. Click **+ Upload Document**
3. Choose a category and upload the file
4. Click **Save**

Residents can download these documents at any time.
