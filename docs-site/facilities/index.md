# Facilities & Visitors Overview

This section covers how residents book community amenities and how visitor access is managed.

## Modules

| Module | Description | Link |
|--------|-------------|------|
| **Facility Bookings** | Reserve community spaces like pools, gyms, and meeting rooms | [Facility Bookings](/facilities/bookings) |
| **Visitor Access** | Request and manage visitor entry permits | [Visitor Access](/facilities/visitor-access) |
| **Community Directory** | Browse community amenities and contacts | [Directory](/facilities/directory) |

## Who Can Use These Features?

| Feature | Who Can Use |
|---------|-------------|
| Book a facility | Tenants, Owners |
| Approve facility bookings | Service Managers (Facility Booking type), Admins |
| Request visitor access | Tenants, Owners |
| Approve visitor access | Service Managers (Visitor Access type), Admins |
| Manage facility settings | Admins |
