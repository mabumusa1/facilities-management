# Request Categories

Service requests are organized into **categories** and **sub-categories** to route them to the right professionals and track performance by type.

## Viewing Categories

Navigate to **Settings → Home Services** or **Settings → Common Area Services** to see and manage all categories.

## Category Structure

```
Category (e.g., Plumbing)
└── Sub-Category (e.g., Leaking Pipe)
    └── Permissions & Configuration
```

## Common Home Service Categories

| Category | Example Sub-Categories |
|----------|----------------------|
| Plumbing | Leaking Pipe, Blocked Drain, Water Heater |
| Electrical | Power Outage, Faulty Socket, Light Fitting |
| HVAC | AC Not Cooling, Thermostat Issue, Filter Replacement |
| Appliances | Refrigerator, Washing Machine, Dishwasher |
| Carpentry | Door / Window, Furniture, Flooring |
| Painting | Interior Walls, Touch-up, Full Repaint |
| Pest Control | Insects, Rodents |
| Cleaning | Deep Clean, Move-in/Move-out Clean |

## Sub-Category Permissions

Each sub-category can be configured with specific permissions:

| Permission | Description |
|------------|-------------|
| **Hide Resident Number** | Professional cannot see the resident's phone |
| **Hide Resident Name** | Professional cannot see the resident's name |
| **Attachments Required** | Resident must upload a photo with the request |
| **Allow Professional to Reschedule** | Professional can suggest a new visit time |
| **Manager Can Close** | Manager can close the request without an invoice |
| **Skip Code Verification** | Professional does not need a confirmation code |

These are configured by Admins in **Settings → Home Services**. See [Home Services Settings](/settings/home-services) for details.
