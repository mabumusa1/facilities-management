# Assigning Professionals

Service Managers and Admins are responsible for reviewing incoming requests and assigning them to the appropriate service professional.

## Viewing Unassigned Requests

Navigate to **Requests** and filter by **Status: New** to see all requests awaiting assignment.

The "Requires Attention" section on the Dashboard also highlights unassigned requests.

## Assigning a Professional

1. Open a **New** service request
2. Click **Assign Professional**
3. A dialog appears showing available professionals filtered by:
   - Service category matching the request type
   - Availability (not already overloaded)
4. Select a professional from the list
5. Set a **scheduled date and time** for the visit
6. Add any internal notes for the professional
7. Click **Assign**

The request status changes to **Assigned** and the professional receives a notification.

## Reassigning a Request

If a professional is unavailable or the assignment needs to change:
1. Open the request
2. Click **Reassign**
3. Select a different professional
4. Confirm

## Rejecting a Request

If a request is invalid or outside the scope of service:
1. Open the request
2. Click **Reject**
3. Enter a reason for rejection
4. Click **Confirm**

The requester is notified with the rejection reason.

## Bulk Assignment

To assign multiple requests at once:
1. From the Requests list, select multiple requests using the checkboxes
2. Click **Assign** in the bulk actions bar
3. Choose a professional and scheduled time
4. Confirm

## Professional Capacity

::: tip
Monitor professional workload in the **Contacts → Service Professionals** section. The detail page for each professional shows their current open assignments.
:::

## Closing a Request (Manager Close)

In some categories, managers are permitted to close a request directly without waiting for the professional to submit an invoice:
1. Open the resolved request
2. Click **Close Request**
3. Enter closure notes
4. Confirm

This is available only when configured in the service category settings.
