# Creating a Service Request

Service requests can be submitted by **admins**, **owners**, and **tenants**.

## Creating a Home Service Request

A home service request covers maintenance or repairs inside a unit.

1. Navigate to **Requests → + New Request**
2. Select **Home Service Request**
3. Fill in the details:

| Field | Description |
|-------|-------------|
| **Community / Building / Unit** | The unit where the issue is |
| **Category** | Main category (e.g., Plumbing, Electrical, HVAC) |
| **Sub-Category** | Specific issue type |
| **Title** | Short description of the problem |
| **Description** | Detailed explanation |
| **Priority** | Low, Medium, High, or Urgent |
| **Preferred Date/Time** | When you would like the visit to happen |
| **Photos** | Attach photos of the issue (required for some categories) |

4. Click **Submit**

## Creating a Common Area Service Request

A common area request covers shared spaces and amenities.

1. Navigate to **Requests → + New Request**
2. Select **Common Area Service Request**
3. Select the **Community** and the **affected area** (e.g., Lobby, Pool, Parking)
4. Fill in category, description, and priority
5. Attach photos if available
6. Click **Submit**

## What Happens After Submission

1. The request appears in the **Service Requests** list with status **New**
2. A Service Manager receives a notification
3. The manager reviews and **assigns** a professional
4. You will receive notifications as the status changes

::: tip
You can track the status of your request in real-time under **Requests** in the navigation menu.
:::

## Request Priority Levels

| Priority | Use When |
|----------|----------|
| **Low** | Cosmetic issues or non-urgent improvements |
| **Medium** | Issues affecting comfort but not safety |
| **High** | Issues significantly impacting daily life |
| **Urgent** | Safety hazards, flooding, power outages |

Urgent requests are flagged prominently for immediate manager attention.
