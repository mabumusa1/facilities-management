# Reports Overview

The Reports module provides analytical insights into your property portfolio's performance.

::: info
Reports are accessible to **Admins** and **Accounting Managers**.
:::

## Available Reports

| Report | Description | Link |
|--------|-------------|------|
| **Lease Statistics** | Lease counts, revenue, and trends | [View →](/reports/leases) |
| **Occupancy Report** | Unit occupancy rates over time | [View →](/reports/occupancy) |
| **Maintenance Report** | Service request metrics | [View →](/reports/maintenance) |
| **Rent Collection** | Payment tracking and collection rates | [View →](/reports/rent-collection) |

## Accessing Reports

Navigate to **Reports** in the left menu.

## Filtering Reports

All reports support filtering by:
- **Date range** – select the period to analyze
- **Community** – focus on a specific community
- **Building** – drill down to a specific building

## Exporting Reports

Each report has an **Export** button to download the data as:
- **Excel (.xlsx)** – for further analysis
- **PDF** – for sharing and printing

## Power BI Reports

If Power BI integration is configured for your account, advanced interactive dashboards are available in **Dashboard → System Reports**.
