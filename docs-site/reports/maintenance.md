# Maintenance Report

The Maintenance Report provides analytics on service request volumes, resolution times, and category breakdowns.

## Accessing the Report

Navigate to **Reports → Maintenance**.

## Metrics Available

### Volume
- Total requests submitted in the period
- New requests vs. previous period
- Open vs. resolved requests

### Resolution Time
- Average time from submission to resolution
- Average time to first assignment
- Requests past their target resolution time

### Category Breakdown
- Requests by category (Plumbing, Electrical, etc.)
- Categories with highest volume
- Categories with longest resolution time

### Professional Performance
- Requests per professional
- Average resolution time by professional
- Rejection rate by professional

### Status Distribution
Breakdown of all requests by current status (New, Assigned, In Progress, Resolved, Canceled, Rejected).

## Filters

| Filter | Options |
|--------|---------|
| Date Range | Custom or preset |
| Community / Building | All or specific |
| Category | All or specific |
| Request Type | Home Services, Common Area |
| Professional | All or specific professional |

## Using This Report

- **High volume in a specific category** (e.g., Plumbing) may indicate systemic infrastructure issues
- **Long average assignment time** suggests the need for more professionals or better scheduling
- **High rejection rate** may indicate unclear request categories or professional capacity issues
