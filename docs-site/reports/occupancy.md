# Occupancy Report

The Occupancy Report tracks how many units are occupied versus available over time.

## Accessing the Report

Navigate to **Reports → Occupancy**.

## Metrics Available

### Overall Occupancy Rate
Percentage of units currently rented:
```
Occupancy Rate = (Rented Units / Total Units) × 100
```

### Vacancy
- Number of units currently available
- Average vacancy duration
- Units vacant for more than 30/60/90 days

### Trend
- Month-by-month occupancy rate chart
- Seasonal patterns (if enough historical data)

### Breakdown
- Occupancy by community
- Occupancy by building
- Occupancy by unit type (Studio, 1BR, 2BR, etc.)

## Filters

| Filter | Options |
|--------|---------|
| Date Range | Custom or preset periods |
| Community | All or specific |
| Building | All or specific |
| Unit Type | Studio, 1BR, 2BR, Villa, Commercial, etc. |

## Using This Report

- **Below 85% occupancy** is a signal to review pricing or marketing
- **High vacancy duration** on specific unit types may indicate pricing mismatch
- **Seasonal dips** help plan proactive lease renewal campaigns
