# Dashboard

The dashboard is your home base after logging in. It provides a real-time overview of your property portfolio.

## Dashboard Widgets

### Requires Attention
A prioritized list of items needing immediate action:
- Leases expiring soon
- Overdue rent payments
- Pending service request assignments
- Unreviewed lease applications

Click any item to navigate directly to the relevant record.

### Units Overview
A summary of all units across your portfolio:

| Status | Description |
|--------|-------------|
| **Available** | Units ready to be leased or sold |
| **Rented** | Units with an active lease |
| **Sold** | Units that have been sold |
| **Sold & Rented** | Units sold but still occupied by a tenant |

### Leases Summary
- Total active leases
- Leases expiring within 30/60/90 days
- Draft leases awaiting activation

### Service Requests
- Open requests by status
- Requests assigned vs. unassigned
- Average resolution time

### Marketplace
- Active listings
- Pending bookings
- Recent offers

### Financial Overview
- Total rent collected this month
- Outstanding balances
- Recent transactions

### Facilities Utilization
- Upcoming facility bookings
- Pending booking approvals

### Visitors
- Visitor access requests for today
- Pending approvals

## Navigating the System

The main navigation menu is on the left sidebar (desktop) or accessible via the menu icon (mobile):

```
Dashboard
Properties  →  Communities, Buildings, Units
Leasing     →  Leases, Applications, Sub-Leases
Requests    →  Service Requests
Contacts    →  Owners, Tenants, Admins, Professionals
Accounting  →  Transactions, Chart of Accounts
Reports     →  Analytics and Reports
Settings    →  System Configuration
```

## Notifications

The bell icon (🔔) in the top navigation shows unread notifications. Click it to view all notifications. You can configure which events trigger notifications in **More → Notification Preferences**.

## User Profile

Click your name or avatar in the top-right corner to:
- Edit your profile information
- Change your password
- Configure notification preferences
- Log out
