# Announcements

**Announcements** are community-wide messages posted by administrators to inform residents about events, maintenance, policy changes, or any community news.

## Viewing Announcements

Residents and owners can view all active announcements from **Dashboard → Announcements** or the **Announcements** section in the navigation.

## Creating an Announcement

::: info
Only users with the **Marketing Manager** or **Admin** role can create announcements.
:::

1. Navigate to **Announcements → + New Announcement**
2. Fill in:

| Field | Description |
|-------|-------------|
| **Title** | Short, descriptive headline |
| **Body** | Full announcement text |
| **Community** | Which community (or All) |
| **Publish Date** | When the announcement goes live |
| **Expiry Date** | When it's automatically removed |
| **Attach Image** | Optional banner image |

3. Click **Save as Draft** or **Publish Now**

## Announcement Statuses

| Status | Meaning |
|--------|---------|
| **Draft** | Saved but not visible to residents |
| **Published** | Live and visible |
| **Expired** | Past the expiry date; auto-hidden |
| **Canceled** | Manually unpublished |

## Editing an Announcement

Open the announcement and click **Edit**. All fields can be updated while in **Draft** status. Published announcements can have their expiry date updated or be canceled.

## Canceling an Announcement

To immediately hide an active announcement:
1. Open the announcement
2. Click **Cancel**
3. Confirm

## Announcement Statistics

Click on any announcement to see engagement statistics:
- Number of views
- View trend over time

This helps you understand which types of announcements residents engage with most.
