# Owners

**Owners** are individuals or entities that own one or more units or properties in the portfolio.

## Viewing Owners

Navigate to **Contacts → Owners** to see all owner records. Use the search bar to find an owner by name, email, or phone.

## Adding an Owner

1. Navigate to **Contacts → Owners → + New Owner**
2. Fill in the owner's information:

| Field | Required |
|-------|----------|
| First Name | ✅ |
| Last Name | ✅ |
| Email | ✅ |
| Phone | ✅ |
| National ID | ✅ |
| Nationality | Optional |
| Address | Optional |

3. Click **Save**

## Owner Details

Click an owner's name to open their record. The detail page shows:

### Overview
- Personal information
- Contact details

### Units Tab
All units this owner holds. Click any unit to open its details.

### Leases Tab
Lease contracts associated with this owner's units.

### Transactions Tab
Financial records tied to this owner.

### Documents Tab
Uploaded files (ID copies, ownership certificates, etc.)

## Editing an Owner

Open the owner record and click **Edit**. Update the fields and click **Save**.

## Linking an Owner to a Unit

Ownership is established when a unit is assigned to an owner during unit creation or through the unit's edit form. Navigate to the unit and set the **Owner** field to link them.
