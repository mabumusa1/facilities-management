# Home Services Configuration

Configure the categories, sub-categories, and rules that govern how home service requests work in your community.

## Accessing Home Services Settings

Navigate to **Settings → Home Services**.

## Managing Categories

### Adding a Category

1. Click **+ New Category**
2. Enter the **Category Name** (e.g., Plumbing, Electrical)
3. Add an **icon** or color label
4. Select which **communities** this category is available in (or All)
5. Click **Save**

### Adding Sub-Categories

1. Open a category
2. Click **+ New Sub-Category**
3. Enter the **Sub-Category Name** (e.g., Leaking Pipe)
4. Configure permissions:

| Permission | Description |
|------------|-------------|
| **Hide Resident Number** | Don't show tenant's phone to the professional |
| **Hide Resident Name** | Don't show tenant's name to the professional |
| **Require Attachments** | Resident must attach a photo when submitting |
| **Allow Professional to Reschedule** | Professional can propose a new date/time |
| **Manager Can Close** | Allow managers to close without invoice |
| **Skip Code Verification** | No confirmation code required on-site |
| **Skip Photo Requirement** | Professional not required to upload completion photos |

5. Click **Save**

### Editing or Deactivating a Category

Open the category and click **Edit** to modify its name, scope, or sub-categories. Set the status to **Inactive** to hide it from residents without deleting it.

## Community Scope

Each service category can be enabled for:
- **All communities** in your account
- **Specific communities** – select from the list

This allows you to offer different services in different properties.
