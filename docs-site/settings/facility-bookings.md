# Facility Booking Settings

Configure how residents book community facilities.

## Accessing Facility Booking Settings

Navigate to **Settings → Facility Bookings**.

## General Settings

| Setting | Description |
|---------|-------------|
| **Enable Facility Bookings** | Turn the booking system on or off |
| **Require Approval** | Whether bookings need manager approval |
| **Advance Booking Window** | How many days in advance a booking can be made |
| **Max Booking Duration** | Maximum hours per booking |
| **Max Concurrent Bookings** | How many active bookings a tenant can have at once |
| **Cancellation Policy** | How many hours before the booking it can be canceled |

## Per-Facility Settings

Each facility can have its own booking rules. Open a facility from **Settings → Facilities** or from the **Directory** and configure:

- **Operating Hours** – the time window when bookings are allowed
- **Capacity** – maximum attendees
- **Pricing** – if the facility has a booking fee
- **Blackout Dates** – dates when the facility is closed (maintenance, events)
- **Booking Contract** – if required before confirmation

## Notification Settings

Configure notifications for:
- New booking requests
- Booking approvals or rejections
- Upcoming booking reminders
- Cancellations
