# Visitor Request Settings

Configure how visitor access requests work in your community.

## Accessing Visitor Settings

Navigate to **Settings → Visitor Requests**.

## Configuration Options

| Setting | Description |
|---------|-------------|
| **Enable Visitor Access** | Turn the visitor access feature on or off |
| **Require Approval** | Whether all visitor requests need manager approval or are auto-approved |
| **Auto-Approve Window** | If auto-approving, how far in advance can requests be submitted |
| **Allowed Entry Hours** | Restrict what times visitors can enter |
| **Max Visitors per Request** | Limit the number of visitors per permit |
| **Required Fields** | Which visitor fields are mandatory |

## Community-Specific Settings

Visitor access rules can be configured per community. Open the community settings and navigate to the **Visitor Requests** tab to set community-specific rules.

## Notification Settings

Configure who receives notifications when:
- A visitor request is submitted
- A request is approved or rejected
- A visitor checks in or out
