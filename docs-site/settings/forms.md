# Forms Builder

The **Forms Builder** allows Admins to create custom forms that residents fill out when submitting service requests or lease applications.

## Accessing Forms

Navigate to **Settings → Forms**.

## Creating a Form

1. Click **+ New Form**
2. Select the **community** and **building** the form applies to
3. Give the form a **name**
4. Add fields by clicking **+ Add Field**:

### Available Field Types

| Field Type | Use For |
|------------|---------|
| **Text** | Short text answers |
| **Textarea** | Long descriptions |
| **Number** | Numeric values |
| **Date** | Date picker |
| **Dropdown** | Select from a list of options |
| **Checkbox** | Yes/No or multiple selection |
| **File Upload** | Allow document or photo uploads |

5. For each field, set:
   - **Label** – what the user sees
   - **Required** – whether the field must be filled
   - **Placeholder** – hint text

6. Drag and drop fields to reorder them
7. Click **Preview** to see how the form will look
8. Click **Save**

## Assigning Forms to Service Categories

After creating a form, you can attach it to a service sub-category:
1. Go to **Settings → Home Services**
2. Open the sub-category
3. Select the **Custom Form** from the dropdown
4. Save

When residents submit a request in that sub-category, they will see the custom form.

## Editing a Form

Open the form and click **Edit**. All fields can be added, removed, or reordered. Existing submissions are not affected by changes to the form.
