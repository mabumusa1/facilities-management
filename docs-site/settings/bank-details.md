# Bank Details

Configure the bank account information that appears on invoices and payment instructions sent to tenants.

## Accessing Bank Settings

Navigate to **Settings → Bank Details**.

## Adding a Bank Account

1. Click **+ Add Bank Account**
2. Fill in:

| Field | Description |
|-------|-------------|
| **Bank Name** | Name of the bank |
| **Account Name** | Name on the bank account |
| **Account Number** | Bank account number |
| **IBAN** | International Bank Account Number |
| **SWIFT / BIC** | Bank routing code for international transfers |
| **Currency** | Currency of the account |
| **Branch** | Branch name or code |

3. Click **Save**

## Multiple Accounts

You can add multiple bank accounts (e.g., for different currencies or communities). Mark one as **Default** to use it on all standard invoices.

## Displaying Bank Details on Invoices

Bank details are automatically included on generated invoices when configured. The account shown on an invoice matches the community's currency.

::: warning
Keep bank account details accurate. Incorrect IBAN or account numbers will cause payment failures. Always verify with your bank before updating.
:::
