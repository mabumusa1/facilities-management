# Settings Overview

The **Settings** section is where Admins configure the system behavior, service categories, forms, and financial settings.

::: info
Settings are only accessible to users with the **Admin** role.
:::

## Available Settings

| Section | Description | Link |
|---------|-------------|------|
| **Home Services** | Configure maintenance categories and sub-categories | [View →](/settings/home-services) |
| **Visitor Requests** | Configure visitor access rules and defaults | [View →](/settings/visitor-requests) |
| **Facility Bookings** | Configure booking rules and approval workflows | [View →](/settings/facility-bookings) |
| **Forms Builder** | Create custom forms for requests and applications | [View →](/settings/forms) |
| **Invoice Templates** | Customize invoice appearance | [View →](/settings/invoices) |
| **Bank Details** | Add payment account information | [View →](/settings/bank-details) |

## Accessing Settings

Navigate to **Settings** in the left menu.
