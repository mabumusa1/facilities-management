# Overdue Tracking

The **Overdues** section helps you identify and follow up on unpaid rent and fees.

## Accessing Overdue Records

Navigate to **Accounting → Overdues** to see all charges that are past their due date.

## What You'll See

The overdue list shows:
- **Tenant name** and unit
- **Amount due** and original due date
- **Days overdue**
- **Lease reference**

You can filter by:
- Community / Building
- Number of days overdue (e.g., 30+, 60+, 90+)
- Amount range

## Actions for Overdue Accounts

| Action | Description |
|--------|-------------|
| **Record Payment** | Mark the overdue charge as paid |
| **Send Reminder** | Send a payment reminder notification to the tenant |
| **View Lease** | Open the full lease record for context |
| **View Tenant** | Open the tenant contact record |

## Tenant Account View

Navigate to **Accounting → Tenant Accounts** for a per-tenant summary of all outstanding and paid balances. This view shows:
- Running balance per tenant
- Payment history
- All unpaid charges with due dates

## Automated Reminders

::: tip
Configure automated payment reminders in **Settings** to reduce manual follow-up work. The system can automatically send reminder emails a specified number of days before and after the due date.
:::
