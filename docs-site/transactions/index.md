# Transactions Overview

The Transactions module tracks all financial activity in your portfolio — rent payments, fees, expenses, and more.

## Accessing Transactions

Navigate to **Accounting** in the left menu. The main view shows recent transactions with summary cards at the top:
- **Total Money In** – income this period
- **Total Money Out** – expenses this period
- **Outstanding** – unpaid charges

## Transaction Types

| Type | Description |
|------|-------------|
| **Money In** | Rent payments, fees collected, deposits received |
| **Money Out** | Expenses, contractor payments, refunds |

## Transaction Statuses

| Status | Meaning |
|--------|---------|
| **Paid** | Payment has been received/made |
| **Due** | Payment is expected but not yet received |
| **Overdue** | Payment was due but has not been received |
