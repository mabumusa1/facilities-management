# Recording Transactions

## Recording a Manual Transaction

Use this to record a rent payment received, a fee charged, or an expense paid.

1. Navigate to **Accounting → + Record Transaction**
2. Fill in the details:

| Field | Description |
|-------|-------------|
| **Type** | Money In (income) or Money Out (expense) |
| **Category** | Transaction category (e.g., Rent, Service Charge, Insurance) |
| **Sub-Category** | More specific classification |
| **Amount** | Transaction amount |
| **Currency** | Defaults to the community currency |
| **Date** | When the transaction occurred |
| **Due Date** | When payment is/was due |
| **Status** | Paid or Due |
| **Linked To** | Associated lease, unit, or contact |
| **Reference** | External reference number (bank transfer, receipt, etc.) |
| **Notes** | Any additional information |

3. Click **Save**

## Recording Rent Payments

Rent payments are typically associated with a lease. To record a rent payment:

1. Open the relevant **Lease**
2. Go to the **Transactions** tab
3. Click **+ Record Payment**
4. Select the charge you're paying (e.g., "January Rent")
5. Enter the amount and payment date
6. Click **Save**

The charge status changes from **Due** to **Paid**.

## Bulk Payment Recording

To record payments for multiple tenants at once:
1. Navigate to **Accounting → Money In**
2. Click **Bulk Payment**
3. Select the leases/charges you're marking as paid
4. Enter the payment date
5. Confirm

## Transaction Detail View

Click any transaction to see:
- Full details and history
- Linked lease and tenant
- Payment receipts or attachments
- Any associated journal entries

## Editing or Deleting a Transaction

::: warning
Transactions that are linked to active leases or have been reconciled should not be deleted. Contact your administrator if a correction is needed.
:::

Minor corrections (e.g., adding a reference number) can be made by opening the transaction and clicking **Edit**.
