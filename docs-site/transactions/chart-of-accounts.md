# Chart of Accounts

The **Chart of Accounts** is the foundation of your financial structure — a hierarchical list of account categories used to classify all transactions.

## Accessing the Chart of Accounts

Navigate to **Accounting → Chart of Accounts**.

## Account Structure

Accounts are organized in a hierarchy:

```
Account Type (e.g., Revenue)
└── Category (e.g., Rental Income)
    └── Sub-Category (e.g., Residential Rent)
```

## Default Account Types

| Type | Usage |
|------|-------|
| **Revenue** | Money received from tenants and operations |
| **Expense** | Costs and operational expenses |
| **Asset** | Property values, deposits held |
| **Liability** | Amounts owed to others |

## Viewing Transactions by Account

Click any account in the chart to see all transactions classified under that account.

## Adding Custom Categories

To add a new transaction category:
1. Navigate to **Settings** (contact your administrator if you don't see this option)
2. Navigate to the relevant transaction category section
3. Click **+ Add Category**
4. Enter the name and select the parent account type
5. Save

::: info
Category structure changes affect all future transaction recording. Coordinate with your accounting team before adding new categories.
:::
