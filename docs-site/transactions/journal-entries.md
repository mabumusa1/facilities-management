# Journal Entries

**Journal entries** are manual double-entry accounting records used for adjustments, corrections, and non-standard transactions.

::: info
Journal entries are an advanced accounting feature, typically used by Accounting Managers.
:::

## Accessing Journal Entries

Navigate to **Accounting → Journal Entries**.

## Creating a Journal Entry

1. Click **+ New Journal Entry**
2. Enter:
   - **Date** – the accounting date of the entry
   - **Description** – what the entry represents
   - **Reference** – any external reference number
3. Add **Debit** and **Credit** lines:
   - Select the account for each line
   - Enter the amount
   - Debits must equal Credits (the system enforces this)
4. Click **Save**

## Common Use Cases

| Scenario | Journal Entry |
|----------|--------------|
| Correcting a mis-categorized transaction | Dr: Correct Account / Cr: Original Account |
| Recording depreciation | Dr: Depreciation Expense / Cr: Accumulated Depreciation |
| Recognizing a deferred payment | Dr: Unearned Revenue / Cr: Revenue |
| Writing off bad debt | Dr: Bad Debt Expense / Cr: Accounts Receivable |

## Viewing and Editing Journal Entries

All journal entries are listed in **Accounting → Journal Entries**. Click any entry to view the full debit/credit breakdown. Entries can be edited or reversed (a new counter-entry is created automatically to maintain the audit trail).
