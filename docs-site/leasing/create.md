# Creating a Lease

This guide walks you through drafting and activating a new lease contract.

## Before You Begin

Make sure the following are in place:
- The **unit** you want to lease has **Available** status
- The **tenant** exists in the system (or you will create them during the wizard)
- You have the **Sales & Leasing Manager** or **Admin** role

## Lease Creation Wizard

Navigate to **Leasing → + New Lease**. The wizard walks you through several steps.

### Step 1: Select Unit

1. Choose the **Community** and **Building**
2. Select the **Unit** from the list of available units
3. Click **Next**

### Step 2: Tenant Information

Choose an existing tenant or create a new one:

**Existing Tenant:** Search by name, email, or phone number and select.

**New Tenant:** Fill in:

| Field | Required |
|-------|----------|
| First Name | ✅ |
| Last Name | ✅ |
| Email | ✅ |
| Phone Number | ✅ |
| National ID | ✅ |
| Nationality | Optional |

Click **Next**.

### Step 3: Lease Terms

| Field | Description |
|-------|-------------|
| **Contract Number** | Auto-generated or enter manually |
| **Start Date** | When the lease begins |
| **End Date** | When the lease expires |
| **Rental Amount** | Monthly rent |
| **Payment Frequency** | Monthly, Quarterly, Annually, etc. |
| **Security Deposit** | Deposit amount (if applicable) |
| **Grace Period (days)** | Days after due date before late fees apply |

### Step 4: Additional Fees

Add any recurring charges on top of rent:
- Service charges
- Parking fees
- Utility fees

Click **+ Add Fee**, choose the category, and enter the amount.

### Step 5: Documents & Attachments

Upload supporting documents:
- Signed lease agreement (PDF)
- Tenant ID copy
- Any other required documents

### Step 6: Review & Save

Review all the details. Click **Save as Draft** to save without activating, or **Activate** to make the lease immediately active.

## Activating a Draft Lease

If you saved the lease as a draft:
1. Open the lease from **Leasing → Leases**
2. Review the details
3. Click **Activate**
4. Confirm the action in the dialog

Once activated, the unit status changes to **Rented** and the tenant can access their lease details.

::: tip
The lease can also be set to **auto-activate** on its start date if you leave it in Draft status.
:::
