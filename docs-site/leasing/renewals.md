# Lease Renewals

Renewing a lease keeps a tenant in their unit without creating a new contract from scratch. All history is preserved.

## Finding Leases to Renew

Navigate to **Leasing → Expiring Leases** to see all leases approaching their end date. You can filter by:
- Expiring in 30 / 60 / 90 days
- Community or building
- Tenant name

## Renewing a Lease

1. Open the lease you want to renew (from the expiring leases list or the main leases list)
2. Click **Renew**
3. Update the lease terms for the new period:

| Field | Description |
|-------|-------------|
| **New End Date** | When the renewed lease expires |
| **New Rental Amount** | Updated monthly rent (if changing) |
| **Payment Schedule** | Any changes to payment frequency |
| **Additional Terms** | Any new clauses or conditions |

4. Optionally upload a new signed contract
5. Click **Confirm Renewal**

The existing lease is updated with the new end date, and a renewal record is added to the lease history.

## Renewal History

Open any lease and go to the **History** tab to see all previous renewal records, including the old and new terms for each renewal.

## What Happens if a Lease Expires Without Renewal?

If no renewal action is taken, the lease status changes to **Expired** on the end date. The unit remains linked to the expired lease for record-keeping, but the unit status returns to **Available** so a new lease can be created.

::: tip
Set up **notification alerts** to receive reminders when leases are approaching expiry. Go to your Notification Preferences to configure this.
:::
