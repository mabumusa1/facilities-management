# Leasing Overview

The Leasing module manages the full lifecycle of rental contracts — from drafting and activation to renewal and termination.

## Lease Lifecycle

```
Draft  →  Active  →  Expired
                 →  Canceled (terminated)
                 →  Closed (move-out)
```

| Status | Description |
|--------|-------------|
| **Draft** | Lease is created but not yet activated |
| **Active** | Lease is in effect; tenant is occupying the unit |
| **Expired** | Lease term ended without renewal |
| **Canceled** | Lease was terminated early |
| **Closed** | Tenant has moved out and the lease is closed |

## Accessing Leasing

Navigate to **Leasing** in the left menu. You will see:
- **All Leases** – searchable and filterable list
- **Applications** – pending lease applications
- **Sub-Leases** – sub-tenancy arrangements
- **Expiring** – leases due for renewal
- **Overdue** – leases with outstanding payments
- **Statistics** – analytics dashboard
