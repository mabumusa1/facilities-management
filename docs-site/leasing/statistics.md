# Leasing Statistics

The **Statistics** page provides a bird's-eye view of your leasing activity.

## Accessing Statistics

Navigate to **Leasing → Statistics**.

## Available Metrics

### Lease Overview
- Total number of leases by status
- New leases created this period vs. previous period
- Leases expiring soon (30/60/90 days)

### Occupancy Rate
- Percentage of units currently rented
- Trend over time (month-by-month chart)
- Breakdown by community and building

### Revenue
- Total contracted rent across active leases
- Collected vs. outstanding amounts
- Average rent per unit type

### Renewals
- Renewal rate (percentage of expiring leases renewed)
- Average renewal duration

## Filtering

Use the **date range** picker and **community/building** filter to narrow statistics to a specific portfolio segment or time period.

## Exporting

Click **Export** to download the statistics as an Excel or PDF report for sharing with stakeholders.
