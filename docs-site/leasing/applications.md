# Lease Applications

Lease applications allow prospective tenants to express interest before a formal lease is created. Applications go through a review and approval workflow.

## Application Workflow

```
Submitted  →  Under Review  →  Approved  →  Quote Sent  →  Converted to Lease
                           →  Rejected
                           →  On Hold  →  Resumed
```

## Viewing Applications

Navigate to **Leasing → Applications** to see all pending and historical applications.

## Submitting an Application (Admin on behalf of prospect)

1. Go to **Leasing → Applications → + New Application**
2. Fill in applicant details:
   - Personal information (name, email, phone, national ID)
   - Preferred unit(s)
   - Desired move-in date
   - Budget
3. Attach supporting documents (ID copy, proof of income, etc.)
4. Click **Submit for Review**

## Reviewing an Application

1. Open the application from the list
2. Review the applicant's information and documents
3. Choose an action:

| Action | Description |
|--------|-------------|
| **Approve** | Move the application forward |
| **Reject** | Decline the application with a reason |
| **Hold** | Pause the application temporarily |
| **Request More Info** | Ask the applicant for additional documents |

## Sending a Quote

After approval, send the prospective tenant a quote outlining:
- The unit and lease terms
- Monthly rent and fees
- Deposit requirements

1. Click **Send Quote**
2. Enter the lease terms for this applicant
3. Click **Send**

The applicant receives the quote by email or in-app notification.

## Converting to a Lease

When the applicant accepts the quote:
1. Open the approved application
2. Click **Convert to Lease**
3. The system pre-fills the [lease creation wizard](/leasing/create) with the application data
4. Review and activate the lease

## Resuming a Held Application

Open the application and click **Resume** to move it back to the active review queue.
